# Skin-Care Product Recommendation System

A production-ready prototype for personalized skincare product recommendations.

## Quick Start

### Backend
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Project Structure
- `/backend` - FastAPI application with ML models
- `/frontend` - React application with Vite
- `/docs` - Documentation and design files
- `/tests` - Test suites

For detailed project information, see `project-details.md`.
